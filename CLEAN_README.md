# ingelap.com.co — Landing page (limpio)

Este README es una versión limpia y única del documento principal. Usa `README.md` en la raíz para información extendida.

Proyecto estático: landing para servicios de instalaciones eléctricas, CCTV, control de acceso y sistemas de detección de incendio.

Cómo probar localmente

```bash
open index.html
# o
python3 -m http.server 8000
# abrir http://localhost:8000
```
